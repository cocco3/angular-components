# Color Primitives

Foundational colors to be used by different themes. These colors are not
context-specific, and should not be used directly in components.

All colors are sourced from the [Figma colors file](https://www.figma.com/design/tLul5JhVTYQ3evZpMEpKEm/Quail?node-id=5225-371288&node-type=frame&t=HjkBV1AN9gHTnK44-0).

<div>
  {Object.values(primitives).map(item => (
    <>
      <h2>{item.name}</h2>
      <div class="docs-colorSwatch sb-unstyled theme-light">
        {Object.entries(item.values).map(([weight, hex]) => {
          const variable = `${item.id}-${weight}`
          return (
            <div class="docs-colorItem" key={variable}>
              <div class="color" style={{ backgroundColor: hex }} />
              <div class="info">
                <div class="variable">{variable}</div>
                <div class="hex">{hex}</div>
              </div>
            </div>
          )
        })}
      </div>
    </>
  ))}
</div>

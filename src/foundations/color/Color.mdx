# Color Primitives

Foundational colors to be used by different themes. These colors are not
context-specific, and should not be used direclty in components.

<div>
  {primitiveColorsArray.map(item => (
    <>
      <h2>{item.name}</h2>
      <div class="sb-colorSwatch">
        {Object.entries(item.colors).map(([weight, hex]) => {
          const variable = `${item.id}-${weight}`
          return (
            <div class="sb-unstyled theme-light item" key={variable}>
              <div class="color" style={{ backgroundColor: hex }} />
              <div class="info">
                <div>{variable}</div>
                <div class="hex">{hex}</div>
              </div>
            </div>
          )
        })}
      </div>
    </>
  ))}
</div>

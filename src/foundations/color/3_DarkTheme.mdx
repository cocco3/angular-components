# Dark theme

Color variables that are defined based on their intended use or meaning. They
provide context and convey information, such as status, state, or action.

You can also opt into it by adding the `theme-dark` class to any container.

All variables are sourced from the [Figma colors file](https://www.figma.com/design/tLul5JhVTYQ3evZpMEpKEm/Quail?node-id=6472-97620&node-type=frame&t=HjkBV1AN9gHTnK44-0).

<div>
  {Object.values(darkTheme).map(item => (
    <>
      <h2>{item.name}</h2>
      <div class="docs-colorSet sb-unstyled theme-light">
        {Object.entries(item.values).map(([weight, hex]) => {
          const variable = `${item.id}-${weight}`
          return (
            <div class="docs-colorItem" key={variable}>
              <div class="color" style={{ backgroundColor: hex }} />
              <div class="info">
                <div class="variable">{variable}</div>
                <div class="hex">{hex}</div>
              </div>
            </div>
          )
        })}
      </div>
    </>
  ))}
</div>
